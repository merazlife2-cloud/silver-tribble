# Photo2Profit Working Upload App

Corrected app build with:
- Real Product Maker upload button
- Real Scanner take/upload photo button
- Photo2Profit navy, blush pink, white, and gold branding
- Separate AI Scanner and Product Maker tools
- ZIP-code pricing
- Yard Sale / Local Thrift / Fast Online / Stretch Online prices
- BUY / MAYBE / SKIP scanner math
- White product stage preview
- Mini storefront
- Cart and checkout
- Shipping and local pickup
- Seller dashboard

Important: this version supports image upload and white-stage preview. True AI background removal requires adding a background-removal API/model next.

Run:
npm install
npm run dev

Build:
npm run build

Vercel:
Framework Preset: Vite
Install Command: npm install
Build Command: npm run build
Output Directory: dist
