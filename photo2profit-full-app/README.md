# Photo2Profit Full App

A working Vite + React + TypeScript app for Photo2Profit.

## Built screens
- Home command center
- AI Scanner
- AI Product Maker
- Mini Storefront
- Cart + Checkout
- Seller Dashboard
- Orders
- Draft Queue
- Saved Scans

## Pricing engine
Default pricing mode is Local Thrift. Users can enter ZIP code and switch between:
- Yard Sale
- Local Thrift
- Fast Online Resale
- Stretch Online

Scanner uses the same pricing logic, but adds:
- Buy cost
- Estimated fees
- Net payout
- Net profit
- ROI
- BUY / MAYBE / SKIP recommendation

## Run locally

```bash
npm install
npm run dev
```

## Deploy
Upload this project to GitHub, then import it into Vercel.
